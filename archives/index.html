C++дҪңиҖ…пјҡBjarne Stroustrup

C++жҳҜдёҖдёӘеӨҡиҢғејҸзҡ„иҜӯиЁҖпјӣеҢ…жӢ¬ж”ҜжҢҒйқўеҗ‘еҜ№иұЎпјҢд№ҹиў«жңүдәәз§°дёәвҖңC with ClassesвҖқпјӣ

CFrontпјҡC++еҺҹе§Ӣзҡ„зј–иҜ‘еҷЁпјҢдҪңз”ЁжҠҠC++д»Јз ҒиҪ¬жҲҗCд»Јз Ғпјӣдёәе•ҘеҸ«CFrontпјҹеңЁCзҡ„зј–иҜ‘еҷЁеүҚйқўеҠ дәҶдёҖдёӘзј–иҜ‘еҷЁпјҢжҠҠC++д»Јз ҒйҖҡиҝҮCFrontиҪ¬жҲҗCд»Јз ҒпјҢз„¶еҗҺз”ЁCзј–иҜ‘еҷЁеҺ»зј–иҜ‘з”ҹжҲҗзЁӢеәҸ

йқўеҗ‘еҜ№иұЎпјҡ

й—®йўҳпјҡC++дёӯзҡ„й«ҳзә§зү№жҖ§еҰӮдҪ•еңЁCдёӯиЎЁзӨәпјҹ

  • зұ»йҮҢйқўзҡ„еҮҪж•°зңҹзҡ„еұһдәҺиҝҷдёӘClassеҗ—пјҹsizeof()
  • зұ»дёӯзҡ„thisжҖҺд№ҲжқҘзҡ„пјҹиҜӯжі•зі–иҖҢе·ІпјӣжңүдәӣиҜӯиЁҖдјҡжҳҫзӨәеҶҷselfпјҲPythonгҖҒRustпјү
  • зұ»и°ғз”ЁжҲҗе‘ҳеҮҪж•°жҳҜжҖҺд№ҲеӣһдәӢпјҹиҜӯжі•зі–пјҢе’Ңи°ғз”Ёжҷ®йҖҡе…ЁеұҖеҮҪж•°е№¶жІЎжңүд»Җд№ҲеҢәеҲ«

C++зҡ„иҷҡеҮҪж•°гҖҒJavaзҡ„InterfaceпјҢRustзҡ„Trait

зҲ¶зұ»е®ҡд№үжҺҘеҸЈпјҢз„¶еҗҺеӯҗзұ»жңүдёҚеҗҢзҡ„е®һзҺ°пјҢеңЁе®һйҷ…иҝҗиЎҢзҡ„ж—¶еҖҷдјҡи°ғз”ЁдёҚеҗҢзҡ„еӯҗзұ»е®һзҺ°пјҢ
дёӢеӣҫдёӯзҡ„жҺҘеҸЈеҮҪж•°жң¬иҙЁдёҠжҳҜдёҖдёӘеҮҪж•°жҢҮй’ҲеҸҳйҮҸпјҢеңЁдҪҝз”Ёж—¶жҢҮе®ҡдәҶдёҚеҗҢзҡ„еӯҗзұ»зҡ„е®һзҺ°зҡ„еҮҪж•°

  • зӣ®еҪ•зҡ„е‘ҪеҗҚ
  • жәҗд»Јз Ғж–Үд»¶зҡ„е‘ҪеҗҚ

дҫқиө–еҖ’зҪ®еҺҹеҲҷпјҡ

е®ҡд№үпјҡй«ҳеұӮдёҚеә”иҜҘдҫқиө–дҪҺеұӮпјҢиҰҒдҫқиө–дәҺжҠҪиұЎпјҢдёҚиҰҒдҫқиө–дәҺе…·дҪ“е®һзҺ°пјҲйқўеҗ‘жҺҘеҸЈзј–зЁӢпјү

й—®йўҳпјҡдҫқиө–дәҺжҠҪиұЎпјҹиҝҷйҮҢзҡ„жҠҪиұЎжҢҮзҡ„жҳҜд»Җд№Ҳпјҹ

    • жҲ‘жЎҶжһ¶жҠҠиҝҷдёӘз»“жһ„дҪ“жҢҮй’Ҳdataз»ҷдҪ пјҡдҪ иҙҹиҙЈжҠҠеҜ№еә”зӮ№еҮ»зҡ„еқҗж ҮеЎ«иҝӣеҺ»пјҢз„¶еҗҺжҲ‘жЎҶжһ¶еҶ…йғЁиҰҒз”ЁпјҲдј йҖ’еҲ°еҜ№еә”жҺ§д»¶зҡ„зӮ№еҮ»еӣһи°ғеҮҪж•°пјү

йҖҡиҝҮе®һзҺ°еӣҫзүҮи§Јз ҒеҷЁпјҢе®һзҺ°еҜ№еә”еҮҪж•°еӣһи°ғгҖӮжӯҘйӘӨеҗҢдёҠвҖҰвҖҰ

зҺ°е®һз”ҹжҙ»пјҡе…¬еҸёеҒҡйЎ№зӣ®зҡ„дҫӢеӯҗ

  • дҫқиө–еҸҚиҪ¬еүҚпјҡжҲ‘йўҶеҜјдҫқиө–дәҺжҲ‘пјҢжҲ‘зҡ„д»»еҠЎе®ҢжҲҗдәҶпјҢд»–зҡ„йЎ№зӣ®жүҚиғҪз®—е®ҢжҲҗ
  • дҫқиө–еҸҚиҪ¬еҗҺпјҡжҲ‘йўҶеҜјжҸҗеҮәж ҮеҮҶпјҢжҸҗдҫӣеҜ№еә”зҡ„иө„жәҗпјӣжҲ‘жӢҝеҲ°ж ҮеҮҶе’Ңиө„жәҗпјҢз„¶еҗҺеҺ»е®һзҺ°

дёҖејҖе§ӢйўҶеҜјдҫқиө–дәҺжҲ‘зҡ„е·ҘдҪңпјҢд»–зҡ„д»»еҠЎе°ұз®—е®ҢжҲҗпјӣжҲ‘дҫқиө–дәҺжҲ‘йўҶеҜјз»ҷзҡ„ж ҮеҮҶе’Ңиө„жәҗеҺ»е®ҢжҲҗиҝҷдёӘд»»еҠЎпјӣеҸҰеӨ–зҡ„дёҖдёӘеҸҳеҢ–жҳҜпјҡеҜ№дәҺеүҚиҖ…пјҢдҪ зҡ„йўҶеҜјеҸҜиғҪдјҡжҺ§еҲ¶дҪ зҡ„з»ҶиҠӮпјҲдёҚиғҪз”ЁиҝҷдёӘпјҢеҫ—з”ЁйӮЈдёӘпјӣдёҚиғҪиҝҷж ·жҗһпјҢиҰҒйӮЈж ·жҗһпјүпјӣеҜ№дәҺеҗҺиҖ…пјҢжҲ‘йўҶеҜје°ұжҲҗдәҶж ҮеҮҶзҡ„еҲ¶е®ҡиҖ…

жҠҪиұЎзҡ„е®ҡд№үпјҡ

  • йңҖиҰҒеҜ№дёҡеҠЎжңүи¶іеӨҹзҡ„дәҶи§ЈпјҢеҜ№й—®йўҳжҠҪиұЎ
    • еӣһи°ғдёӯзҡ„еҸӮж•°пјҡиғҪеӨҹе®һзҺ°жӢ“еұ•пјӣдёҚиҰҒеҮәзҺ°еҶ—дҪҷзҡ„еҸӮж•°
  • жҠҪиұЎж ҮеҮҶзҡ„иҰҒжұӮ
    • Androidзҡ„UIзәҝзЁӢпјҡжӯ»еҫӘзҺҜдёҚеҒңзҡ„еҺ»и°ғеӣһи°ғпјҢз”ЁжҲ·еҶҷзҡ„uiд»Јз ҒпјҢз”ЁжҲ·еҶҷзҡ„дәӢд»¶д»Јз Ғпјӣиҝҷдәӣд»Јз Ғеҫ—ж»Ўи¶ідёҖе®ҡзҡ„иҰҒжұӮпјҢжҜ”еҰӮиҜҙпјҡдёҚиғҪеҚЎеӨӘд№…пјӣдҪ еҚЎдҪҸдәҶпјҢжҲ‘еҶ…йғЁзҡ„жӯ»еҫӘзҺҜе°ұиў«иҝҷдёӘеҚЎжӯ»

жҖ§иғҪжҢҮж ҮжҳҜд»Җд№Ҳпјҹ

жҖ§иғҪеҲҶжһҗпјҢе…¶е®һе°ұжҳҜжүҫеҮәеә”з”ЁжҲ–зі»з»ҹзҡ„з“¶йўҲпјҢе№¶и®ҫжі•еҺ»йҒҝе…ҚжҲ–иҖ…зј“и§Је®ғд»¬

  • йҖүжӢ©жҢҮж ҮиҜ„дј°еә”з”ЁзЁӢеәҸе’Ңзі»з»ҹзҡ„жҖ§иғҪпјӣ
  • дёәеә”з”ЁзЁӢеәҸе’Ңзі»з»ҹи®ҫзҪ®жҖ§иғҪзӣ®ж Үпјӣ
  • иҝӣиЎҢжҖ§иғҪеҹәеҮҶжөӢиҜ•пјӣ
  • жҖ§иғҪеҲҶжһҗе®ҡдҪҚз“¶йўҲпјӣ
  • дјҳеҢ–зі»з»ҹе’Ңеә”з”ЁзЁӢеәҸпјӣ
  • жҖ§иғҪзӣ‘жҺ§е’Ңе‘ҠиӯҰгҖӮ

  • иҷҪз„¶зі»з»ҹзҡ„еҺҹзҗҶеҫҲйҮҚиҰҒпјҢдҪҶеңЁеҲҡејҖе§ӢдёҖе®ҡдёҚиҰҒиҜ•еӣҫжҠ“дҪҸжүҖжңүзҡ„е®һзҺ°з»ҶиҠӮ

  • иҫ№еӯҰиҫ№е®һи·өпјҢйҖҡиҝҮеӨ§йҮҸзҡ„жЎҲдҫӢжј”д№ жҺҢжҸЎ Linux жҖ§иғҪзҡ„еҲҶжһҗе’ҢдјҳеҢ–
  • еӢӨжҖқиҖғпјҢеӨҡеҸҚжҖқпјҢе–„жҖ»з»“пјҢеӨҡй—®дёәд»Җд№Ҳ

е№іеқҮиҙҹиҪҪ

е№іеқҮиҙҹиҪҪжҳҜжҢҮеҚ•дҪҚж—¶й—ҙеҶ…пјҢзі»з»ҹеӨ„дәҺеҸҜиҝҗиЎҢзҠ¶жҖҒе’ҢдёҚеҸҜдёӯж–ӯзҠ¶жҖҒзҡ„е№іеқҮиҝӣзЁӢж•°пјҢд№ҹе°ұжҳҜе№іеқҮжҙ»и·ғиҝӣзЁӢж•°пјҢе®ғе’Ң CPU дҪҝз”ЁзҺҮе№¶жІЎжңүзӣҙжҺҘе…ізі»

еҸҜиҝҗиЎҢзҠ¶жҖҒпјҡжҳҜжҢҮжӯЈеңЁдҪҝз”Ё CPU жҲ–иҖ…жӯЈеңЁзӯүеҫ… CPU зҡ„иҝӣзЁӢпјҢд№ҹе°ұжҳҜжҲ‘д»¬еёёз”Ё ps е‘Ҫд»ӨзңӢеҲ°зҡ„пјҢеӨ„дәҺ R зҠ¶жҖҒпјҲRunning жҲ– Runnableпјүзҡ„иҝӣзЁӢ

дёҚеҸҜдёӯж–ӯзҠ¶жҖҒзҡ„иҝӣзЁӢпјҡжҳҜжӯЈеӨ„дәҺеҶ…ж ёжҖҒе…ій”®жөҒзЁӢдёӯзҡ„иҝӣзЁӢпјҢе№¶дё”иҝҷдәӣжөҒзЁӢжҳҜдёҚеҸҜжү“ж–ӯзҡ„пјҢжҜ”еҰӮжңҖеёёи§Ғзҡ„жҳҜзӯүеҫ…зЎ¬д»¶и®ҫеӨҮзҡ„ I/O е“Қеә”пјҢд№ҹе°ұжҳҜжҲ‘д»¬еңЁ ps е‘Ҫд»ӨдёӯзңӢеҲ°зҡ„ D зҠ¶жҖҒпјҲUninterruptible SleepпјҢд№ҹз§°дёә Disk Sleepпјүзҡ„иҝӣзЁӢ

еҪ“ж•°еҖјдёә2ж—¶пјҢд»ЈиЎЁпјҡ

  • еңЁеҸӘжңү 2 дёӘ CPU зҡ„зі»з»ҹдёҠпјҢж„Ҹе‘ізқҖжүҖжңүзҡ„ CPU йғҪеҲҡеҘҪиў«е®Ңе…ЁеҚ з”ЁгҖӮ
  • еңЁ 4 дёӘ CPU зҡ„зі»з»ҹдёҠпјҢж„Ҹе‘ізқҖ CPU жңү 50% зҡ„з©әй—ІгҖӮ
  • иҖҢеңЁеҸӘжңү 1 дёӘ CPU зҡ„зі»з»ҹдёӯпјҢеҲҷж„Ҹе‘ізқҖжңүдёҖеҚҠзҡ„иҝӣзЁӢз«һдәүдёҚеҲ° CPUгҖӮ

  • mpstatпјҡ жҳҜдёҖдёӘеёёз”Ёзҡ„еӨҡж ё CPU жҖ§иғҪеҲҶжһҗе·Ҙе…·пјҢз”ЁжқҘе®һж—¶жҹҘзңӢжҜҸдёӘ CPU зҡ„жҖ§иғҪжҢҮж ҮпјҢд»ҘеҸҠжүҖжңү CPU зҡ„е№іеқҮжҢҮж ҮгҖӮ
  • pidstatпјҡ жҳҜдёҖдёӘеёёз”Ёзҡ„иҝӣзЁӢжҖ§иғҪеҲҶжһҗе·Ҙе…·пјҢз”ЁжқҘе®һж—¶жҹҘзңӢиҝӣзЁӢзҡ„ CPUгҖҒеҶ…еӯҳгҖҒI/O д»ҘеҸҠдёҠдёӢж–ҮеҲҮжҚўзӯүжҖ§иғҪжҢҮж ҮгҖӮ

еңәжҷҜдёҖпјҡCPU еҜҶйӣҶеһӢиҝӣзЁӢ

uptimeпјҡ1 еҲҶй’ҹзҡ„е№іеқҮиҙҹиҪҪдјҡж…ўж…ўеўһеҠ еҲ° 1.00пјҢmpstatпјҡжӯЈеҘҪжңүдёҖдёӘ CPU зҡ„дҪҝз”ЁзҺҮдёә 100%пјҢдҪҶе®ғзҡ„ iowait еҸӘжңү 0гҖӮиҝҷиҜҙжҳҺпјҢе№іеқҮиҙҹиҪҪзҡ„еҚҮй«ҳжӯЈжҳҜз”ұдәҺ CPU дҪҝз”ЁзҺҮдёә 100% гҖӮ

еңәжҷҜдәҢпјҡI/O еҜҶйӣҶеһӢиҝӣзЁӢ

еңәжҷҜдёүпјҡеӨ§йҮҸиҝӣзЁӢзҡ„еңәжҷҜ

8 дёӘиҝӣзЁӢеңЁдәүжҠў 2 дёӘ CPUпјҢжҜҸдёӘиҝӣзЁӢзӯүеҫ… CPU зҡ„ж—¶й—ҙпјҲд№ҹе°ұжҳҜд»Јз Ғеқ—дёӯзҡ„ %wait еҲ—пјүй«ҳиҫҫ 75%гҖӮиҝҷдәӣи¶…еҮә CPU и®Ўз®—иғҪеҠӣзҡ„иҝӣзЁӢпјҢжңҖз»ҲеҜјиҮҙ CPU иҝҮиҪҪгҖӮ

ж №жҚ®д»»еҠЎзҡ„дёҚеҗҢпјҢCPU зҡ„дёҠдёӢж–ҮеҲҮжҚўе°ұеҸҜд»ҘеҲҶдёәеҮ дёӘдёҚеҗҢзҡ„еңәжҷҜпјҢд№ҹе°ұжҳҜиҝӣзЁӢдёҠдёӢж–ҮеҲҮжҚўгҖҒзәҝзЁӢдёҠдёӢж–ҮеҲҮжҚўд»ҘеҸҠдёӯж–ӯдёҠдёӢж–ҮеҲҮжҚўгҖӮ

д»Һз”ЁжҲ·жҖҒеҲ°еҶ…ж ёжҖҒзҡ„иҪ¬еҸҳпјҢйңҖиҰҒйҖҡиҝҮзі»з»ҹи°ғз”ЁжқҘе®ҢжҲҗ

  • иҝӣзЁӢдёҠдёӢж–ҮеҲҮжҚўпјҢжҳҜжҢҮд»ҺдёҖдёӘиҝӣзЁӢеҲҮжҚўеҲ°еҸҰдёҖдёӘиҝӣзЁӢиҝҗиЎҢгҖӮ
  • иҖҢзі»з»ҹи°ғз”ЁиҝҮзЁӢдёӯдёҖзӣҙжҳҜеҗҢдёҖдёӘиҝӣзЁӢеңЁиҝҗиЎҢгҖӮ

  • е…¶дәҢпјҢиҝӣзЁӢеңЁзі»з»ҹиө„жәҗдёҚи¶іпјҲжҜ”еҰӮеҶ…еӯҳдёҚи¶іпјүж—¶пјҢиҰҒзӯүеҲ°иө„жәҗж»Ўи¶іеҗҺжүҚеҸҜд»ҘиҝҗиЎҢпјҢиҝҷдёӘж—¶еҖҷиҝӣзЁӢд№ҹдјҡиў«жҢӮиө·пјҢе№¶з”ұзі»з»ҹи°ғеәҰе…¶д»–иҝӣзЁӢиҝҗиЎҢгҖӮ

  • еҪ“иҝӣзЁӢеҸӘжңүдёҖдёӘзәҝзЁӢж—¶пјҢеҸҜд»Ҙи®ӨдёәиҝӣзЁӢе°ұзӯүдәҺзәҝзЁӢгҖӮ

    • з”ЁжҲ· CPU дҪҝз”ЁзҺҮпјҢеҢ…жӢ¬з”ЁжҲ·жҖҒ CPU дҪҝз”ЁзҺҮпјҲuserпјүе’ҢдҪҺдјҳе…Ҳзә§з”ЁжҲ·жҖҒ CPU дҪҝз”ЁзҺҮпјҲniceпјүпјҢиЎЁзӨә CPU еңЁз”ЁжҲ·жҖҒиҝҗиЎҢзҡ„ж—¶й—ҙзҷҫеҲҶжҜ”гҖӮз”ЁжҲ· CPU дҪҝз”ЁзҺҮй«ҳпјҢйҖҡеёёиҜҙжҳҺжңүеә”з”ЁзЁӢеәҸжҜ”иҫғз№ҒеҝҷгҖӮ

    • зі»з»ҹ CPU дҪҝз”ЁзҺҮпјҢиЎЁзӨә CPU еңЁеҶ…ж ёжҖҒиҝҗиЎҢзҡ„ж—¶й—ҙзҷҫеҲҶжҜ”пјҲдёҚеҢ…жӢ¬дёӯж–ӯпјүгҖӮзі»з»ҹ CPU дҪҝз”ЁзҺҮй«ҳпјҢиҜҙжҳҺеҶ…ж ёжҜ”иҫғз№ҒеҝҷгҖӮ

    • зӯүеҫ… I/O зҡ„ CPU дҪҝз”ЁзҺҮпјҢйҖҡеёёд№ҹз§°дёә iowaitпјҢиЎЁзӨәзӯүеҫ… I/O зҡ„ж—¶й—ҙзҷҫеҲҶжҜ”гҖӮiowait й«ҳпјҢйҖҡеёёиҜҙжҳҺзі»з»ҹдёҺзЎ¬д»¶и®ҫеӨҮзҡ„ I/O дәӨдә’ж—¶й—ҙжҜ”иҫғй•ҝгҖӮ

    • иҪҜдёӯж–ӯе’ҢзЎ¬дёӯж–ӯзҡ„ CPU дҪҝз”ЁзҺҮпјҢеҲҶеҲ«иЎЁзӨәеҶ…ж ёи°ғз”ЁиҪҜдёӯж–ӯеӨ„зҗҶзЁӢеәҸгҖҒзЎ¬дёӯж–ӯеӨ„зҗҶзЁӢеәҸзҡ„ж—¶й—ҙзҷҫеҲҶжҜ”гҖӮе®ғд»¬зҡ„дҪҝз”ЁзҺҮй«ҳпјҢйҖҡеёёиҜҙжҳҺзі»з»ҹеҸ‘з”ҹдәҶеӨ§йҮҸзҡ„дёӯж–ӯгҖӮ

    • йҷӨдәҶдёҠйқўиҝҷдәӣпјҢиҝҳжңүеңЁиҷҡжӢҹеҢ–зҺҜеўғдёӯдјҡз”ЁеҲ°зҡ„зӘғеҸ– CPU дҪҝз”ЁзҺҮпјҲstealпјүе’Ңе®ўжҲ· CPU дҪҝз”ЁзҺҮпјҲguestпјүпјҢеҲҶеҲ«иЎЁзӨәиў«е…¶д»–иҷҡжӢҹжңәеҚ з”Ёзҡ„ CPU ж—¶й—ҙзҷҫеҲҶжҜ”пјҢе’ҢиҝҗиЎҢе®ўжҲ·иҷҡжӢҹжңәзҡ„ CPU ж—¶й—ҙзҷҫеҲҶжҜ”гҖӮ

  • еә”иҜҘжҳҜе№іеқҮиҙҹиҪҪпјҲLoad AverageпјүпјҢд№ҹе°ұжҳҜзі»з»ҹзҡ„е№іеқҮжҙ»и·ғиҝӣзЁӢж•°

  • иҝӣзЁӢдёҠдёӢж–ҮеҲҮжҚў

    • ж— жі•иҺ·еҸ–иө„жәҗиҖҢеҜјиҮҙзҡ„иҮӘж„ҝдёҠдёӢж–ҮеҲҮжҚўпјӣ
    • иў«зі»з»ҹејәеҲ¶и°ғеәҰеҜјиҮҙзҡ„йқһиҮӘж„ҝдёҠдёӢж–ҮеҲҮжҚўгҖӮ

  • CPU зј“еӯҳзҡ„е‘ҪдёӯзҺҮ

жҖ§иғҪе·Ҙе…·

з¬¬дёҖдёӘз»ҙеәҰпјҢд»Һ CPU зҡ„жҖ§иғҪжҢҮж ҮеҮәеҸ‘гҖӮд№ҹе°ұжҳҜиҜҙпјҢеҪ“дҪ иҰҒжҹҘзңӢжҹҗдёӘжҖ§иғҪжҢҮж Үж—¶пјҢиҰҒжё…жҘҡзҹҘйҒ“е“Әдәӣе·Ҙе…·еҸҜд»ҘеҒҡеҲ°гҖӮ

з¬¬дәҢдёӘз»ҙеәҰпјҢд»Һе·Ҙе…·еҮәеҸ‘гҖӮд№ҹе°ұжҳҜеҪ“дҪ е·Із»Ҹе®үиЈ…дәҶжҹҗдёӘе·Ҙе…·еҗҺпјҢиҰҒзҹҘйҒ“иҝҷдёӘе·Ҙе…·иғҪжҸҗдҫӣе“ӘдәӣжҢҮж ҮгҖӮ

еҰӮдҪ•иҝ…йҖҹеҲҶжһҗ CPU зҡ„жҖ§иғҪз“¶йўҲ

жғіеј„жё…жҘҡжҖ§иғҪжҢҮж Үзҡ„е…іиҒ”жҖ§пјҢе°ұиҰҒйҖҡжҷ“жҜҸз§ҚжҖ§иғҪжҢҮж Үзҡ„е·ҘдҪңеҺҹзҗҶ

дёҫдёӘдҫӢеӯҗпјҢз”ЁжҲ· CPU дҪҝз”ЁзҺҮй«ҳпјҢжҲ‘д»¬еә”иҜҘеҺ»жҺ’жҹҘиҝӣзЁӢзҡ„з”ЁжҲ·жҖҒиҖҢдёҚжҳҜеҶ…ж ёжҖҒгҖӮеӣ дёәз”ЁжҲ· CPU дҪҝз”ЁзҺҮеҸҚжҳ зҡ„е°ұжҳҜз”ЁжҲ·жҖҒзҡ„ CPU дҪҝз”Ёжғ…еҶөпјҢиҖҢеҶ…ж ёжҖҒзҡ„ CPU дҪҝз”Ёжғ…еҶөеҸӘдјҡеҸҚжҳ еҲ°зі»з»ҹ CPU дҪҝз”ЁзҺҮдёҠгҖӮ

дёәдәҶзј©е°ҸжҺ’жҹҘиҢғеӣҙпјҢжҲ‘йҖҡеёёдјҡе…ҲиҝҗиЎҢеҮ дёӘж”ҜжҢҒжҢҮж ҮиҫғеӨҡзҡ„е·Ҙе…·пјҢеҰӮ topгҖҒvmstat е’Ң pidstat

жҖ§иғҪдјҳеҢ–ж–№жі•и®ә

  • йҰ–е…ҲпјҢж—ўз„¶иҰҒеҒҡжҖ§иғҪдјҳеҢ–пјҢйӮЈиҰҒжҖҺд№ҲеҲӨж–ӯе®ғжҳҜдёҚжҳҜжңүж•Ҳе‘ўпјҹзү№еҲ«жҳҜдјҳеҢ–еҗҺпјҢеҲ°еә•иғҪжҸҗеҚҮеӨҡе°‘жҖ§иғҪе‘ўпјҹ

  • з¬¬дәҢпјҢжҖ§иғҪй—®йўҳйҖҡеёёдёҚжҳҜзӢ¬з«Ӣзҡ„пјҢеҰӮжһңжңүеӨҡдёӘжҖ§иғҪй—®йўҳеҗҢж—¶еҸ‘з”ҹпјҢдҪ еә”иҜҘе…ҲдјҳеҢ–е“ӘдёҖдёӘе‘ўпјҹ

  • з¬¬дёүпјҢжҸҗеҚҮжҖ§иғҪзҡ„ж–№жі•е№¶дёҚжҳҜе”ҜдёҖзҡ„пјҢеҪ“жңүеӨҡз§Қж–№жі•еҸҜд»ҘйҖүжӢ©ж—¶пјҢдҪ дјҡйҖүз”Ёе“ӘдёҖз§Қе‘ўпјҹжҳҜдёҚжҳҜжҖ»йҖүйӮЈдёӘжңҖеӨ§зЁӢеәҰжҸҗеҚҮжҖ§иғҪзҡ„ж–№жі•е°ұиЎҢдәҶе‘ўпјҹ

жҖҺд№ҲиҜ„дј°жҖ§иғҪдјҳеҢ–зҡ„ж•Ҳжһңпјҹ

  • зЎ®е®ҡжҖ§иғҪзҡ„йҮҸеҢ–жҢҮж ҮгҖӮ
  • жөӢиҜ•дјҳеҢ–еүҚзҡ„жҖ§иғҪжҢҮж ҮгҖӮ
  • жөӢиҜ•дјҳеҢ–еҗҺзҡ„жҖ§иғҪжҢҮж ҮгҖӮ

жҲ‘зҡ„е»әи®®жҳҜдёҚиҰҒеұҖйҷҗеңЁеҚ•дёҖз»ҙеәҰзҡ„жҢҮж ҮдёҠпјҢдҪ иҮіе°‘иҰҒд»Һеә”з”ЁзЁӢеәҸе’Ңзі»з»ҹиө„жәҗиҝҷдёӨдёӘз»ҙеәҰпјҢеҲҶеҲ«йҖүжӢ©дёҚеҗҢзҡ„жҢҮж ҮгҖӮжҜ”еҰӮпјҢд»Ҙ Web еә”з”ЁдёәдҫӢпјҡ

  • еә”з”ЁзЁӢеәҸзҡ„з»ҙеәҰпјҢжҲ‘д»¬еҸҜд»Ҙз”ЁеҗһеҗҗйҮҸе’ҢиҜ·жұӮе»¶иҝҹжқҘиҜ„дј°еә”з”ЁзЁӢеәҸзҡ„жҖ§иғҪгҖӮ
  • зі»з»ҹиө„жәҗзҡ„з»ҙеәҰпјҢжҲ‘д»¬еҸҜд»Ҙз”Ё CPU дҪҝз”ЁзҺҮжқҘиҜ„дј°зі»з»ҹзҡ„ CPU дҪҝз”Ёжғ…еҶөгҖӮ

еӨҡдёӘжҖ§иғҪй—®йўҳеҗҢж—¶еӯҳеңЁпјҢиҰҒжҖҺд№ҲйҖүжӢ©пјҹ

еңЁжҖ§иғҪжөӢиҜ•зҡ„йўҶеҹҹпјҢжөҒдј еҫҲе№ҝзҡ„дёҖдёӘиҜҙжі•жҳҜвҖңдәҢе…«еҺҹеҲҷвҖқпјҢд№ҹе°ұжҳҜиҜҙ 80% зҡ„й—®йўҳйғҪжҳҜз”ұ 20% зҡ„д»Јз ҒеҜјиҮҙзҡ„гҖӮеҸӘиҰҒжүҫеҮәиҝҷ 20% зҡ„дҪҚзҪ®пјҢдҪ е°ұеҸҜд»ҘдјҳеҢ– 80% зҡ„жҖ§иғҪгҖӮжүҖд»ҘпјҢжҲ‘жғіиЎЁиҫҫзҡ„жҳҜпјҢе№¶дёҚжҳҜжүҖжңүзҡ„жҖ§иғҪй—®йўҳйғҪеҖјеҫ—дјҳеҢ–гҖӮ

  • з¬¬дёҖпјҢеҰӮжһңеҸ‘зҺ°жҳҜзі»з»ҹиө„жәҗиҫҫеҲ°дәҶз“¶йўҲпјҢжҜ”еҰӮ CPU дҪҝз”ЁзҺҮиҫҫеҲ°дәҶ 100%пјҢйӮЈд№ҲйҰ–е…ҲдјҳеҢ–зҡ„дёҖе®ҡжҳҜзі»з»ҹиө„жәҗдҪҝз”Ёй—®йўҳгҖӮе®ҢжҲҗзі»з»ҹиө„жәҗз“¶йўҲзҡ„дјҳеҢ–еҗҺпјҢжҲ‘д»¬жүҚиҰҒиҖғиҷ‘е…¶д»–й—®йўҳгҖӮ
  • з¬¬дәҢпјҢй’ҲеҜ№дёҚеҗҢзұ»еһӢзҡ„жҢҮж ҮпјҢйҰ–е…ҲеҺ»дјҳеҢ–йӮЈдәӣз”ұз“¶йўҲеҜјиҮҙзҡ„пјҢжҖ§иғҪжҢҮж ҮеҸҳеҢ–е№…еәҰжңҖеӨ§зҡ„й—®йўҳгҖӮжҜ”еҰӮдә§з”ҹз“¶йўҲеҗҺпјҢз”ЁжҲ· CPU дҪҝз”ЁзҺҮеҚҮй«ҳдәҶ 10%пјҢиҖҢзі»з»ҹ CPU дҪҝз”ЁзҺҮеҚҙеҚҮй«ҳдәҶ 50%пјҢиҝҷдёӘж—¶еҖҷе°ұеә”иҜҘйҰ–е…ҲдјҳеҢ–зі»з»ҹ CPU зҡ„дҪҝз”ЁгҖӮ

еҘҪзҡ„еә”з”ЁзЁӢеәҸжҳҜжҖ§иғҪдјҳеҢ–зҡ„жңҖз»Ҳзӣ®зҡ„е’Ңз»“жһңпјҢзі»з»ҹдјҳеҢ–жҖ»жҳҜдёәеә”з”ЁзЁӢеәҸжңҚеҠЎзҡ„гҖӮжүҖд»ҘпјҢеҝ…йЎ»иҰҒдҪҝз”Ёеә”з”ЁзЁӢеәҸзҡ„жҢҮж ҮпјҢжқҘиҜ„дј°жҖ§иғҪдјҳеҢ–зҡ„ж•ҙдҪ“ж•ҲжһңгҖӮзі»з»ҹиө„жәҗзҡ„дҪҝз”Ёжғ…еҶөжҳҜеҪұе“Қеә”з”ЁзЁӢеәҸжҖ§иғҪзҡ„ж №жәҗгҖӮжүҖд»ҘпјҢйңҖиҰҒз”Ёзі»з»ҹиө„жәҗзҡ„жҢҮж ҮпјҢжқҘи§ӮеҜҹе’ҢеҲҶжһҗз“¶йўҲзҡ„жқҘжәҗгҖӮ

жңүеӨҡз§ҚдјҳеҢ–ж–№жі•ж—¶пјҢиҰҒеҰӮдҪ•йҖүжӢ©?

жҖ§иғҪдјҳеҢ–е№¶йқһжІЎжңүжҲҗжң¬гҖӮжҖ§иғҪдјҳеҢ–йҖҡеёёдјҡеёҰжқҘеӨҚжқӮеәҰзҡ„жҸҗеҚҮпјҢйҷҚдҪҺзЁӢеәҸзҡ„еҸҜз»ҙжҠӨжҖ§пјҢиҝҳеҸҜиғҪеңЁдјҳеҢ–дёҖдёӘжҢҮж Үж—¶пјҢеј•еҸ‘е…¶д»–жҢҮж Үзҡ„ејӮеёёгҖӮд№ҹе°ұжҳҜиҜҙпјҢеҫҲеҸҜиғҪдҪ дјҳеҢ–дәҶдёҖдёӘжҢҮж ҮпјҢеҸҰдёҖдёӘжҢҮж Үзҡ„жҖ§иғҪеҚҙеҸҳе·®дәҶгҖӮ

дёҖдёӘеҫҲе…ёеһӢзҡ„дҫӢеӯҗжҳҜжҲ‘е°ҶеңЁзҪ‘з»ңйғЁеҲҶи®ІеҲ°зҡ„ DPDKпјҲData Plane Development KitпјүгҖӮDPDK жҳҜдёҖз§ҚдјҳеҢ–зҪ‘з»ңеӨ„зҗҶйҖҹеәҰзҡ„ж–№жі•пјҢе®ғйҖҡиҝҮз»•ејҖеҶ…ж ёзҪ‘з»ңеҚҸи®®ж Ҳзҡ„ж–№жі•пјҢжҸҗеҚҮзҪ‘з»ңзҡ„еӨ„зҗҶиғҪеҠӣгҖӮдёҚиҝҮе®ғжңүдёҖдёӘеҫҲе…ёеһӢзҡ„иҰҒжұӮпјҢе°ұжҳҜиҰҒзӢ¬еҚ дёҖдёӘ CPU д»ҘеҸҠдёҖе®ҡж•°йҮҸзҡ„еҶ…еӯҳеӨ§йЎөпјҢе№¶дё”жҖ»жҳҜд»Ҙ 100% зҡ„ CPU дҪҝз”ЁзҺҮиҝҗиЎҢгҖӮжүҖд»ҘпјҢеҰӮжһңдҪ зҡ„ CPU ж ёж•°еҫҲе°‘пјҢе°ұжңүзӮ№еҫ—дёҚеҒҝеӨұдәҶгҖӮ

CPU дјҳеҢ–

еҰӮдҪ•жүҚиғҪйҷҚдҪҺ CPU дҪҝз”ЁзҺҮпјҢжҸҗй«ҳ CPU зҡ„е№¶иЎҢеӨ„зҗҶиғҪеҠӣ

еә”з”ЁзЁӢеәҸдјҳеҢ–

  • зј–иҜ‘еҷЁдјҳеҢ–пјҡеҫҲеӨҡзј–иҜ‘еҷЁйғҪдјҡжҸҗдҫӣдјҳеҢ–йҖүйЎ№пјҢйҖӮеҪ“ејҖеҗҜе®ғд»¬пјҢеңЁзј–иҜ‘йҳ¶ж®өдҪ е°ұеҸҜд»ҘиҺ·еҫ—зј–иҜ‘еҷЁзҡ„её®еҠ©пјҢжқҘжҸҗеҚҮжҖ§иғҪгҖӮжҜ”еҰӮпјҢ gcc е°ұжҸҗдҫӣдәҶдјҳеҢ–йҖүйЎ№ -O2пјҢејҖеҗҜеҗҺдјҡиҮӘеҠЁеҜ№еә”з”ЁзЁӢеәҸзҡ„д»Јз ҒиҝӣиЎҢдјҳеҢ–гҖӮ
  • з®—жі•дјҳеҢ–пјҡдҪҝз”ЁеӨҚжқӮеәҰжӣҙдҪҺзҡ„з®—жі•пјҢеҸҜд»Ҙжҳҫи‘—еҠ еҝ«еӨ„зҗҶйҖҹеәҰгҖӮжҜ”еҰӮпјҢеңЁж•°жҚ®жҜ”иҫғеӨ§зҡ„жғ…еҶөдёӢпјҢеҸҜд»Ҙз”Ё O(nlogn) зҡ„жҺ’еәҸз®—жі•пјҲеҰӮеҝ«жҺ’гҖҒеҪ’е№¶жҺ’еәҸзӯүпјүпјҢд»Јжӣҝ O(n^2) зҡ„жҺ’еәҸз®—жі•пјҲеҰӮеҶ’жіЎгҖҒжҸ’е…ҘжҺ’еәҸзӯүпјүгҖӮ
  • ејӮжӯҘеӨ„зҗҶпјҡдҪҝз”ЁејӮжӯҘеӨ„зҗҶпјҢеҸҜд»ҘйҒҝе…ҚзЁӢеәҸеӣ дёәзӯүеҫ…жҹҗдёӘиө„жәҗиҖҢдёҖзӣҙйҳ»еЎһпјҢд»ҺиҖҢжҸҗеҚҮзЁӢеәҸзҡ„е№¶еҸ‘еӨ„зҗҶиғҪеҠӣгҖӮжҜ”еҰӮпјҢжҠҠиҪ®иҜўжӣҝжҚўдёәдәӢд»¶йҖҡзҹҘпјҢе°ұеҸҜд»ҘйҒҝе…ҚиҪ®иҜўиҖ—иҙ№ CPU зҡ„й—®йўҳгҖӮ
  • еӨҡзәҝзЁӢд»ЈжӣҝеӨҡиҝӣзЁӢпјҡеүҚйқўи®ІиҝҮпјҢзӣёеҜ№дәҺиҝӣзЁӢзҡ„дёҠдёӢж–ҮеҲҮжҚўпјҢзәҝзЁӢзҡ„дёҠдёӢж–ҮеҲҮжҚўе№¶дёҚеҲҮжҚўиҝӣзЁӢең°еқҖз©әй—ҙпјҢеӣ жӯӨеҸҜд»ҘйҷҚдҪҺдёҠдёӢж–ҮеҲҮжҚўзҡ„жҲҗжң¬гҖӮ
  • е–„з”Ёзј“еӯҳпјҡз»Ҹеёёи®ҝй—®зҡ„ж•°жҚ®жҲ–иҖ…и®Ўз®—иҝҮзЁӢдёӯзҡ„жӯҘйӘӨпјҢеҸҜд»Ҙж”ҫеҲ°еҶ…еӯҳдёӯзј“еӯҳиө·жқҘпјҢиҝҷж ·еңЁдёӢж¬Ўз”Ёж—¶е°ұиғҪзӣҙжҺҘд»ҺеҶ…еӯҳдёӯиҺ·еҸ–пјҢеҠ еҝ«зЁӢеәҸзҡ„еӨ„зҗҶйҖҹеәҰгҖӮ

зі»з»ҹдјҳеҢ–

  • CPU з»‘е®ҡпјҡжҠҠиҝӣзЁӢз»‘е®ҡеҲ°дёҖдёӘжҲ–иҖ…еӨҡдёӘ CPU дёҠпјҢеҸҜд»ҘжҸҗй«ҳ CPU зј“еӯҳзҡ„е‘ҪдёӯзҺҮпјҢеҮҸе°‘и·Ё CPU и°ғеәҰеёҰжқҘзҡ„дёҠдёӢж–ҮеҲҮжҚўй—®йўҳгҖӮ
  • CPU зӢ¬еҚ пјҡи·ҹ CPU з»‘е®ҡзұ»дјјпјҢиҝӣдёҖжӯҘе°Ҷ CPU еҲҶз»„пјҢе№¶йҖҡиҝҮ CPU дәІе’ҢжҖ§жңәеҲ¶дёәе…¶еҲҶй…ҚиҝӣзЁӢгҖӮиҝҷж ·пјҢиҝҷдәӣ CPU е°ұз”ұжҢҮе®ҡзҡ„иҝӣзЁӢзӢ¬еҚ пјҢжҚўеҸҘиҜқиҜҙпјҢдёҚе…Ғи®ёе…¶д»–иҝӣзЁӢеҶҚжқҘдҪҝз”Ёиҝҷдәӣ CPUгҖӮ
  • дјҳе…Ҳзә§и°ғж•ҙпјҡдҪҝз”Ё nice и°ғж•ҙиҝӣзЁӢзҡ„дјҳе…Ҳзә§пјҢжӯЈеҖји°ғдҪҺдјҳе…Ҳзә§пјҢиҙҹеҖји°ғй«ҳдјҳе…Ҳзә§гҖӮдјҳе…Ҳзә§зҡ„ж•°еҖјеҗ«д№үеүҚйқўжҲ‘д»¬жҸҗеҲ°иҝҮпјҢеҝҳдәҶзҡ„иҜқеҸҠж—¶еӨҚд№ дёҖдёӢгҖӮеңЁиҝҷйҮҢпјҢйҖӮеҪ“йҷҚдҪҺйқһж ёеҝғеә”з”Ёзҡ„дјҳе…Ҳзә§пјҢеўһй«ҳж ёеҝғеә”з”Ёзҡ„дјҳе…Ҳзә§пјҢеҸҜд»ҘзЎ®дҝқж ёеҝғеә”з”Ёеҫ—еҲ°дјҳе…ҲеӨ„зҗҶгҖӮ
  • дёәиҝӣзЁӢи®ҫзҪ®иө„жәҗйҷҗеҲ¶пјҡдҪҝз”Ё Linux cgroups жқҘи®ҫзҪ®иҝӣзЁӢзҡ„ CPU дҪҝз”ЁдёҠйҷҗпјҢеҸҜд»ҘйҳІжӯўз”ұдәҺжҹҗдёӘеә”з”ЁиҮӘиә«зҡ„й—®йўҳпјҢиҖҢиҖ—е°Ҫзі»з»ҹиө„жәҗгҖӮ
  • NUMAпјҲNon-Uniform Memory AccessпјүдјҳеҢ–пјҡж”ҜжҢҒ NUMA зҡ„еӨ„зҗҶеҷЁдјҡиў«еҲ’еҲҶдёәеӨҡдёӘ nodeпјҢжҜҸдёӘ node йғҪжңүиҮӘе·ұзҡ„жң¬ең°еҶ…еӯҳз©әй—ҙгҖӮNUMA дјҳеҢ–пјҢе…¶е®һе°ұжҳҜи®© CPU е°ҪеҸҜиғҪеҸӘи®ҝй—®жң¬ең°еҶ…еӯҳгҖӮ
  • дёӯж–ӯиҙҹиҪҪеқҮиЎЎпјҡж— и®әжҳҜиҪҜдёӯж–ӯиҝҳжҳҜзЎ¬дёӯж–ӯпјҢе®ғд»¬зҡ„дёӯж–ӯеӨ„зҗҶзЁӢеәҸйғҪеҸҜиғҪдјҡиҖ—иҙ№еӨ§йҮҸзҡ„ CPUгҖӮејҖеҗҜ irqbalance жңҚеҠЎжҲ–иҖ…й…ҚзҪ® smp_affinityпјҢе°ұеҸҜд»ҘжҠҠдёӯж–ӯеӨ„зҗҶиҝҮзЁӢиҮӘеҠЁиҙҹиҪҪеқҮиЎЎеҲ°еӨҡдёӘ CPU дёҠгҖӮ

еҚғдёҮйҒҝе…ҚиҝҮж—©дјҳеҢ–

вҖңиҝҮж—©дјҳеҢ–жҳҜдёҮжҒ¶д№ӢжәҗвҖқ вҖ”вҖ”й«ҳеҫ·зәі

еӣ дёәпјҢдёҖж–№йқўпјҢдјҳеҢ–дјҡеёҰжқҘеӨҚжқӮжҖ§зҡ„жҸҗеҚҮпјҢйҷҚдҪҺеҸҜз»ҙжҠӨжҖ§пјӣеҸҰдёҖж–№йқўпјҢйңҖжұӮдёҚжҳҜдёҖжҲҗдёҚеҸҳзҡ„гҖӮй’ҲеҜ№еҪ“еүҚжғ…еҶөиҝӣиЎҢзҡ„дјҳеҢ–пјҢеҫҲеҸҜиғҪе№¶дёҚйҖӮеә”еҝ«йҖҹеҸҳеҢ–зҡ„ж–°йңҖжұӮгҖӮиҝҷж ·пјҢеңЁж–°йңҖжұӮеҮәзҺ°ж—¶пјҢиҝҷдәӣеӨҚжқӮзҡ„дјҳеҢ–пјҢеҸҚиҖҢеҸҜиғҪйҳ»зўҚж–°еҠҹиғҪзҡ„ејҖеҸ‘гҖӮжүҖд»ҘпјҢжҖ§иғҪдјҳеҢ–жңҖеҘҪжҳҜйҖҗжӯҘе®Ңе–„пјҢеҠЁжҖҒиҝӣиЎҢпјҢдёҚиҝҪжұӮдёҖжӯҘеҲ°дҪҚпјҢиҖҢиҰҒйҰ–е…ҲдҝқиҜҒиғҪж»Ўи¶іеҪ“еүҚзҡ„жҖ§иғҪиҰҒжұӮгҖӮеҪ“еҸ‘зҺ°жҖ§иғҪдёҚж»Ўи¶іиҰҒжұӮжҲ–иҖ…еҮәзҺ°жҖ§иғҪз“¶йўҲж—¶пјҢеҶҚж №жҚ®жҖ§иғҪиҜ„дј°зҡ„з»“жһңпјҢйҖүжӢ©жңҖйҮҚиҰҒзҡ„жҖ§иғҪй—®йўҳиҝӣиЎҢдјҳеҢ–гҖӮ

жҖ»з»“

дҪҶжҳҜи®°дҪҸпјҢдёҖе®ҡиҰҒеҝҚдҪҸвҖңжҠҠ CPU жҖ§иғҪдјҳеҢ–еҲ°жһҒиҮҙвҖқзҡ„еҶІеҠЁпјҢеӣ дёә CPU е№¶дёҚжҳҜе”ҜдёҖзҡ„жҖ§иғҪеӣ зҙ гҖӮеңЁеҗҺз»ӯзҡ„ж–Үз« дёӯпјҢжҲ‘иҝҳдјҡд»Ӣз»ҚжӣҙеӨҡзҡ„жҖ§иғҪй—®йўҳпјҢжҜ”еҰӮеҶ…еӯҳгҖҒзҪ‘з»ңгҖҒI/O з”ҡиҮіжҳҜжһ¶жһ„и®ҫи®Ўзҡ„й—®йўҳгҖӮ

еҶ…еӯҳжҖ§иғҪзҜҮ

еҹәзЎҖзҜҮпјҡLinuxеҶ…еӯҳжҳҜжҖҺд№Ҳе·ҘдҪңзҡ„пјҹ

еҶ…еӯҳжҳ е°„

зү©зҗҶеҶ…еӯҳгҖҒиҷҡжӢҹеҶ…еӯҳ

иҷҡжӢҹең°еқҖз©әй—ҙзҡ„еҶ…йғЁеҸҲиў«еҲҶдёәеҶ…ж ёз©әй—ҙе’Ңз”ЁжҲ·з©әй—ҙ

MMU

еҶ…еӯҳжҳ е°„зҡ„жңҖе°ҸеҚ•дҪҚпјҢд№ҹе°ұжҳҜйЎөпјҢйҖҡеёёжҳҜ 4 KB еӨ§е°ҸгҖӮиҝҷж ·пјҢжҜҸдёҖж¬ЎеҶ…еӯҳжҳ е°„пјҢйғҪйңҖиҰҒе…іиҒ” 4 KB жҲ–иҖ… 4KB ж•ҙж•°еҖҚзҡ„еҶ…еӯҳз©әй—ҙ

иҷҡжӢҹеҶ…еӯҳз©әй—ҙеҲҶеёғ

  • еҸӘиҜ»ж®өпјҢеҢ…жӢ¬д»Јз Ғе’ҢеёёйҮҸзӯү
  • ж•°жҚ®ж®өпјҢеҢ…жӢ¬е…ЁеұҖеҸҳйҮҸзӯү
  • е ҶпјҢеҢ…жӢ¬еҠЁжҖҒеҲҶй…Қзҡ„еҶ…еӯҳпјҢд»ҺдҪҺең°еқҖејҖе§Ӣеҗ‘дёҠеўһй•ҝ
  • ж–Үд»¶жҳ е°„ж®өпјҢеҢ…жӢ¬еҠЁжҖҒеә“гҖҒе…ұдә«еҶ…еӯҳзӯүпјҢд»Һй«ҳең°еқҖејҖе§Ӣеҗ‘дёӢеўһй•ҝ
  • ж ҲпјҢеҢ…жӢ¬еұҖйғЁеҸҳйҮҸе’ҢеҮҪж•°и°ғз”Ёзҡ„дёҠдёӢж–Үзӯү
  • ж Ҳзҡ„еӨ§е°ҸжҳҜеӣәе®ҡзҡ„пјҢдёҖиҲ¬жҳҜ 8 MB

еҶ…еӯҳеҲҶй…ҚдёҺеӣһж”¶

malloc() жҳҜ C ж ҮеҮҶеә“жҸҗдҫӣзҡ„еҶ…еӯҳеҲҶй…ҚеҮҪж•°пјҢеҜ№еә”еҲ°зі»з»ҹи°ғз”ЁдёҠпјҢжңүдёӨз§Қе®һзҺ°ж–№ејҸпјҢеҚі brk() е’Ң mmap()гҖӮ

  • еҜ№е°Ҹеқ—еҶ…еӯҳпјҲе°ҸдәҺ 128KпјүпјҢC ж ҮеҮҶеә“дҪҝз”Ё brk() жқҘеҲҶй…ҚпјҢд№ҹе°ұжҳҜйҖҡиҝҮз§»еҠЁе ҶйЎ¶зҡ„дҪҚзҪ®жқҘеҲҶй…ҚеҶ…еӯҳгҖӮиҝҷдәӣеҶ…еӯҳйҮҠж”ҫеҗҺе№¶дёҚдјҡз«ӢеҲ»еҪ’иҝҳзі»з»ҹпјҢиҖҢжҳҜиў«зј“еӯҳиө·жқҘпјҢиҝҷж ·е°ұеҸҜд»ҘйҮҚеӨҚдҪҝз”ЁгҖӮ
  • иҖҢеӨ§еқ—еҶ…еӯҳпјҲеӨ§дәҺ 128KпјүпјҢеҲҷзӣҙжҺҘдҪҝз”ЁеҶ…еӯҳжҳ е°„ mmap() жқҘеҲҶй…ҚпјҢд№ҹе°ұжҳҜеңЁж–Үд»¶жҳ е°„ж®өжүҫдёҖеқ—з©әй—ІеҶ…еӯҳеҲҶй…ҚеҮәеҺ»гҖӮ

дёӨз§Қж–№ејҸзҡ„дјҳзјәзӮ№пјҡ

  • brk() ж–№ејҸзҡ„зј“еӯҳпјҢеҸҜд»ҘеҮҸе°‘зјәйЎөејӮеёёзҡ„еҸ‘з”ҹпјҢжҸҗй«ҳеҶ…еӯҳи®ҝй—®ж•ҲзҺҮгҖӮдёҚиҝҮпјҢз”ұдәҺиҝҷдәӣеҶ…еӯҳжІЎжңүеҪ’иҝҳзі»з»ҹпјҢеңЁеҶ…еӯҳе·ҘдҪңз№Ғеҝҷж—¶пјҢйў‘з№Ғзҡ„еҶ…еӯҳеҲҶй…Қе’ҢйҮҠж”ҫдјҡйҖ жҲҗеҶ…еӯҳзўҺзүҮгҖӮ
  • иҖҢ mmap() ж–№ејҸеҲҶй…Қзҡ„еҶ…еӯҳпјҢдјҡеңЁйҮҠж”ҫж—¶зӣҙжҺҘеҪ’иҝҳзі»з»ҹпјҢжүҖд»ҘжҜҸж¬Ў mmap йғҪдјҡеҸ‘з”ҹзјәйЎөејӮеёёгҖӮеңЁеҶ…еӯҳе·ҘдҪңз№Ғеҝҷж—¶пјҢйў‘з№Ғзҡ„еҶ…еӯҳеҲҶй…ҚдјҡеҜјиҮҙеӨ§йҮҸзҡ„зјәйЎөејӮеёёпјҢдҪҝеҶ…ж ёзҡ„з®ЎзҗҶиҙҹжӢ…еўһеӨ§гҖӮиҝҷд№ҹжҳҜ malloc еҸӘеҜ№еӨ§еқ—еҶ…еӯҳдҪҝз”Ё mmap зҡ„еҺҹеӣ гҖӮ

еҪ“иҝҷдёӨз§Қи°ғз”ЁеҸ‘з”ҹеҗҺпјҢе…¶е®һе№¶жІЎжңүзңҹжӯЈеҲҶй…ҚеҶ…еӯҳгҖӮиҝҷдәӣеҶ…еӯҳпјҢйғҪеҸӘеңЁйҰ–ж¬Ўи®ҝй—®ж—¶жүҚеҲҶй…ҚпјҢд№ҹе°ұжҳҜйҖҡиҝҮзјәйЎөејӮеёёиҝӣе…ҘеҶ…ж ёдёӯпјҢеҶҚз”ұеҶ…ж ёжқҘеҲҶй…ҚеҶ…еӯҳгҖӮ

еңЁеҸ‘зҺ°еҶ…еӯҳзҙ§еј ж—¶пјҢзі»з»ҹе°ұдјҡйҖҡиҝҮдёҖзі»еҲ—жңәеҲ¶жқҘеӣһж”¶еҶ…еӯҳпјҢжҜ”еҰӮдёӢйқўиҝҷдёүз§Қж–№ејҸпјҡ

  • еӣһж”¶зј“еӯҳпјҢжҜ”еҰӮдҪҝз”Ё LRUпјҲLeast Recently Usedпјүз®—жі•пјҢеӣһж”¶жңҖиҝ‘дҪҝз”ЁжңҖе°‘зҡ„еҶ…еӯҳйЎөйқўпјӣ
  • еӣһж”¶дёҚеёёи®ҝй—®зҡ„еҶ…еӯҳпјҢжҠҠдёҚеёёз”Ёзҡ„еҶ…еӯҳйҖҡиҝҮдәӨжҚўеҲҶеҢәзӣҙжҺҘеҶҷеҲ°зЈҒзӣҳдёӯпјӣ
  • жқҖжӯ»иҝӣзЁӢпјҢеҶ…еӯҳзҙ§еј ж—¶зі»з»ҹиҝҳдјҡйҖҡиҝҮ OOMпјҲOut of MemoryпјүпјҢзӣҙжҺҘжқҖжҺүеҚ з”ЁеӨ§йҮҸеҶ…еӯҳзҡ„иҝӣзЁӢгҖӮ

з¬¬дәҢз§Қж–№ејҸеӣһж”¶дёҚеёёи®ҝй—®зҡ„еҶ…еӯҳж—¶пјҢдјҡз”ЁеҲ°дәӨжҚўеҲҶеҢәпјҲд»ҘдёӢз®Җз§° SwapпјүгҖӮSwap е…¶е®һе°ұжҳҜжҠҠдёҖеқ—зЈҒзӣҳз©әй—ҙеҪ“жҲҗеҶ…еӯҳжқҘз”ЁгҖӮе®ғеҸҜд»ҘжҠҠиҝӣзЁӢжҡӮж—¶дёҚз”Ёзҡ„ж•°жҚ®еӯҳеӮЁеҲ°зЈҒзӣҳдёӯпјҲиҝҷдёӘиҝҮзЁӢз§°дёәжҚўеҮәпјүпјҢеҪ“иҝӣзЁӢи®ҝй—®иҝҷдәӣеҶ…еӯҳж—¶пјҢеҶҚд»ҺзЈҒзӣҳиҜ»еҸ–иҝҷдәӣж•°жҚ®еҲ°еҶ…еӯҳдёӯпјҲиҝҷдёӘиҝҮзЁӢз§°дёәжҚўе…ҘпјүгҖӮ

First, letвҖҷs take a look at the ownership rules. Keep these rules in mind as we work through the examples that illustrate them:

  • Each value in Rust has a variable thatвҖҷs called its owner.

  • When the owner goes out of scope, the value will be dropped.

The ownership and borrowing rules are enforced at compile time!
