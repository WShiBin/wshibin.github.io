Final Cut Pro ​

Edit ​

  • Undo: Command-Z
  • Redo: Command-Shift-Z
  • Enable/Disable Clip: V

Marking ​

  • Range Selection Tool: R

Share and tools ​

  • Share to Default Destination: Command-E
  • Select (Arrow) Tool: A
  • Blade Tool: B
  • Hand Tool: H
  • Trim Tool: T
  • 长按上面的按键，可以临时切换到相应的工具
  • 切了删除左边的部分：alt + [
  • 切了删除右边的部分：alt + ]
  • 删除片段（吸附）：delete
  • 删除片段（留空）：shift + delete
  • Command + B
  • Command + Shift + B

Audio ​

  • 微调：Command + Mouse Drag

View ​

  • Zoom to Fit: Shift-Z
  • 缩放：Command-+/Command--
  • 缩放：选择缩放的区域
  • Zoom Tool: Z
  • 长按上面的按键，可以临时切换到相应的工具，使用一次

Effects ​

  • Paste Attributes: Command-Shift-V

Reference ​

  • https://support.apple.com/zh-hk/guide/final-cut-pro/ver90ba5929/mac
