Gif ​

Overview ​

gif，即 Graphics Interchange Format

格式 ​

所有有意义的文件都有它自己的格式; gif 由以下块组合为一个完成的文件:

  • Header
  • LogicScreenDescriptor
  • Global Color Table
  • Application Extension
  • Graphic Control Extension
  • Plain Text Extension
  • Comment Extension
  • Image Descriptor
  • Local Color Table
  • Image Data
  • Trailer

Header 头部 ​

固定六个字节：

  • 01-03: 固定GIF
  • 04-06: 版本号89a

LogicScreenDescriptor 全局信息 ​

gif 文件的全局信息:

  • 01-02：宽width
  • 03-04：高height
  • 05: 压缩字节
  • 06: 背景色索引
  • 07: 每个像素的宽高比
